This page demonstrates the chat widget loaded directly from your LLM Crafter API server.

Widget is Active!

You should see a chat button in the bottom-right corner. Click it to start chatting!

How to Embed on Your Website

Option 1: Simple Script Tag (Data Attributes)

<script
  src="http://localhost:3000/widget/chat-widget.min.js"
  data-api-key="your-api-key"
  data-agent-id="your-agent-id"
  data-organization-id="your-org-id"
  data-project-id="your-project-id"
  data-api-url="http://localhost:3000"
  data-title="Support Chat"
  data-primary-color="#4a90e2"
></script>

Option 2: Manual Initialization (JavaScript)

<script src="http://localhost:3000/widget/chat-widget.min.js"></script>
<script>
  const widget = new LLMCrafterChatWidget({
    apiKey: 'your-api-key',
    agentId: 'your-agent-id',
    organizationId: 'your-org-id',
    projectId: 'your-project-id',
    apiUrl: 'http://localhost:3000',
    title: 'AI Assistant',
    primaryColor: '#4a90e2'
  });
</script>

Available Widget Files

  • /widget/chat-widget.min.js - Minified (recommended for production)
  • /widget/chat-widget.js - Unminified (for debugging)
  • /widget/chat-widget.umd.js - UMD format
  • /widget/latest.js - Alias for minified version

For Production

Replace http://localhost:3000 with your production API URL:

<script
  src="https://api.yourdomain.com/widget/chat-widget.min.js"
  data-api-key="your-api-key"
  data-agent-id="your-agent-id"
  data-organization-id="your-org-id"
  data-project-id="your-project-id"
  data-api-url="https://api.yourdomain.com"
></script>
