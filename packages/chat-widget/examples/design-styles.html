🎨 LLM Crafter Chat Widget Design Styles

Choose between classic simplicity or modern elegance

🔷 Default Design

Classic, clean, and professional design that works everywhere

  • Circular avatars
  • Rounded message bubbles
  • Soft shadows
  • Roboto font family
  • Fully customizable colors

✨ Modern Design

Contemporary design with smooth animations and refined aesthetics

  • Square-rounded avatars
  • Asymmetric bubble corners
  • Bouncing typing indicator
  • DM Sans font family
  • Scale & fade animations

📝 Implementation Examples

HTML Script Tag (Default Design)
<script src="../dist/chat-widget.min.js" data-api-key="your-api-key" data-agent-id="your-agent-id" data-organization-id="your-org-id" data-project-id="your-project-id" data-design-style="default" data-primary-color="#4a90e2" ></script>
HTML Script Tag (Modern Design)
<script src="../dist/chat-widget.min.js" data-api-key="your-api-key" data-agent-id="your-agent-id" data-organization-id="your-org-id" data-project-id="your-project-id" data-design-style="modern" data-primary-color="#2563eb" ></script>
JavaScript API
const widget = new LLMCrafterChatWidget({ apiKey: 'your-api-key', agentId: 'your-agent-id', organizationId: 'your-org-id', projectId: 'your-project-id', designStyle: 'modern', // 'default' or 'modern' title: 'AI Assistant', primaryColor: '#2563eb', });
💡 Note: Both design styles are fully responsive and support all widget features including streaming, human handoff, message bundling, and conversation persistence. The design style only affects the visual appearance and animations.
